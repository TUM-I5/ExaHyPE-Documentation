Detailed Description

Particle tracing over the DG solver.

Please consult the class FiniteVolumesTracing for details on the semantics of this type. It is an exact analogon for DG.

Update plug-in points

We only update cells which are marked with getHasUpdated(). For enclave solvers, we assume that this flag is only set after a cell has updated. For Runge-Kutta type solvers, I assume that this flag is set by the computation of the final linear combination of estimates.

Return unique action set name.

Returns a description (word) for the mapping which is also used as class name for the generated type. As a consequence, the result should be one word (if possible) and uppercase. Also, every subclass should overwrite this routine.

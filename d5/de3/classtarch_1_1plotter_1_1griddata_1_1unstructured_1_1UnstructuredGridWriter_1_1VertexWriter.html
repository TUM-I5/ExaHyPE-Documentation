Detailed Description

This is the vertex writer you have to create to plot the vertices.

Besides the pure syntax management, the writer also provides a number generator which provides you with a unique id for each vertex.

Please ensure that you call close() on the vertex writer before you close the underlying VTKTextFileWriter.

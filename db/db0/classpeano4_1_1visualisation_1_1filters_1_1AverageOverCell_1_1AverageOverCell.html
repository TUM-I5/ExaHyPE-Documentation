Detailed Description

Run over all cells.

Each cell holds a patch with dof x dof (2d) or dof x dof x dof (3d) cells. Each cell hosts unknown unknowns. This filter reduces each patch to a 1x1 or 1x1x1 outcome, i.e. a point cloud.

Overwrite this one for the particular filter.

We create an empty result array. Then we run over the input sequence of cell_data. Per element, we check if a cell_data to the right within this input sequence overlaps.

List operations are very expensive in Python. I therefore implement a two-pass strategy, where I first sort the patches according to their size. Actually, I sort them according to their inverted size, i.e. big entries come first.

Detailed Description

The default sequence sketches what Peano does if there's no main.

You can alter it. The project holds an instance of this object through its main attribute. So it holds an instance of this class. As a consequence, you can either change this object's attributes or you can replace this object if the object of you choice.

Most people who wanna redefine the main create a subclass of DefaultSequence and overwrite _get_header_file_template() and _get_implementation_file_template(). Some also redefine the default overwrite policy by changing the attribute overwrite.

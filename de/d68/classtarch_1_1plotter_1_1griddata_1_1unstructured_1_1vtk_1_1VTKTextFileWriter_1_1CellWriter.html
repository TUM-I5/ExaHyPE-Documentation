◆ _cellListEntries

int tarch::plotter::griddata::unstructured::vtk::VTKTextFileWriter::CellWriter::_cellListEntries
private

The tag CELLS in a vtk file requires the number of total entries in the following list of cell-interconnection (for a triangle, an entry could look like this: "3 1 2 4", which states that the triangle has 3 links to vetrices with indices 1, 2 and 4.

this makes up four entries). For an unstructured mesh, the element type is not fixed and, hence, the total amount of list entries must be counted by summing up the contributions of each element, when adding the element.

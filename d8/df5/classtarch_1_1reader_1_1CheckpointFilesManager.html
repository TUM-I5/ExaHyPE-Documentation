Detailed Description

A reader class allowing code to load checkpoint subfile if it hasn't be loaded in memory and search in loaded subfiles to find the correct patch data for restarting.

Designed for memory-wise option of restarting, ideally this option will be faster than pure loaded approach and also avold momoery fault in for huge domain.

11class ReconstructPatch(ActionSet):
12 """!
13 This class assumes that you have an NxNxN patch within your block. It also assumes that
14 you have an 2MxNxN patch on your faces. M<N. The code interprets these face-associated
15 data as overlap with the patch, hooks into touchCellLastTime, and projects the cell's
16 patch data onto the overlap (simple copy). See ProjectPatchOntoFaces for details.
17
18 If the face patches are auxiliary patches holding copies from the cell patches, then
19 we can reconstruct N+2M x N+2M x N+2M patches per cell whenever we hit a cell: We
20 create a new temporary array, and copy data from both the cell patch and the faces
21 into this array (gather operation). After that, we can launch the passed functor
22 giving it access to the temporary, large array plus the original patch data.
23 """

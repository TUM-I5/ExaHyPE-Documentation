Detailed Description

Kernel functions for SPH (scalar and vector version).

All constants and kernel coefficients are taken from table 1 of Dehnen & Aly, MNRAS, 425, pp. 1062-1082 (2012).

◆ for() [1/2]

for ( )

We count faces starting with zero: Let the leftmost volume have the index i. i is typically 0 or -1 (if we have halos). When we call this routine for the index i, we first take the volume i and then we average it with i+1. So we look to the right. The result is stored in i. That means, all the indices are kind of shifted by 1/2 to the left. Index (4,3,2) is actually the face located at (4.5, 3, 2).
